MedClaim AI is a full-stack web application built for the IISc Hackathon 2026 (Problem Statement 1). It automates the adjudication of health insurance claims by processing uploaded medical documents (PDFs/images) through a vision AI pipeline and a deterministic rules engine based on Standard Treatment Guidelines (STG).
The system supports all 4 procedure packages — Severe Anemia (MG064A), Cholecystectomy (SG039C), Enteric Fever (MG006A), and Total Knee Replacement (SB039A) — producing a structured per-page JSON output with document presence flags, clinical condition classification, sign assessments, and chronological document ranking.
